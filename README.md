## üöÄ Quick Start

### Option 1: GitHub Pages (Recommended)
Just visit the live version:
```
https://anishvyapari.github.io/tools
```

### Option 2: Local Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/AnishVyapari/tools.git
   cd tools
   ```

2. Open `index.html` in your browser:
   - **Windows**: Double-click the file
   - **Mac/Linux**: Open terminal and run `open index.html`

3. Or start a local server:
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js
   npx http-server
   ```
